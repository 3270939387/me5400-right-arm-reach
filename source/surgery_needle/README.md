Surgery Needle extension (surgery_needle)

This package contains a manager-based RL task template `SurgeryNeedleReach-v0`.
It provides an environment config, minimal MDP helper functions (placeholders),
and an rl_games PPO config to get started training a 7-DoF joint-velocity agent.

Customize the asset prim paths and any environment-specific math in the `mdp.py` helpers.
